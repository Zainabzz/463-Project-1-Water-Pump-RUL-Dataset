# 463-Project-1-Water-Pump-RUL-Dataset
Develop an algorithm‑driven system to segment, cluster, and analyze time‑series sensor data from a real water‑pump machine. The system eses core algo techniques (Divide Conquer seg, Closest Pair, and Maximum Subarray) to characterize machine behavior and relate the results to four machine‑health categories derived from Remaining Useful Life.
